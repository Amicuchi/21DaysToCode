# 👷 Como rodar

```bash

# Clonar o repositório
git clone https://github.com/Amicuchi/100daysOfCode.git

# Entrar numa IDE de sua preferência

# Executar o servidor ou usar um puglin que criar um servidor
```

## 🤔 Como contribuir ? 🤝  <br/>
```bash
- Faça um fork desse repositório;
```
